# THNBOY Thrift

A production-ready premium fashion e-commerce frontend built with **Next.js, TypeScript, Tailwind CSS, Framer Motion, and ShadCN UI**.

## Features

- Premium luxury-inspired UI
- Fully responsive design
- Smooth scroll animations
- Modern product showcase
- Shopping cart interface
- Wishlist UI
- Collections & Categories
- Mystery Bundle page
- Reusable components
- Optimized performance
- SEO-friendly structure

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- ShadCN UI
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the application.
